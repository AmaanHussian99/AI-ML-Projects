# AI & Machine Learning Projects

This repository contains multiple Artificial Intelligence and Machine Learning projects developed using Python, Scikit-Learn, Pandas, NumPy, and data visualization libraries.

## Projects Included

### 1. Click-Through Rate Prediction for Digital Advertising
- Predicts whether a user will click on an online advertisement.
- Uses Logistic Regression for classification.
- Includes data preprocessing, visualization, and model evaluation.

### 2. Netflix Content Clustering and Recommendation System
- Content-based recommendation engine.
- Uses TF-IDF, SVD, K-Means Clustering, and Cosine Similarity.
- Recommends similar movies and TV shows based on content features.

### 3. Heart Disease Prediction Using Machine Learning
- Predicts the likelihood of heart disease using patient health data.
- Implements Logistic Regression, Decision Tree, and Random Forest models.
- Includes performance comparison and ROC-AUC analysis.
